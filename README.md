# DocumentsAndDirectives

Minimal backend scaffolding for the AI TRPG prototype.

## Start

1. Create a virtual environment and install dependencies (FastAPI + Uvicorn).
2. Run the API:

```bash
uvicorn backend.api.main:app --reload
```
