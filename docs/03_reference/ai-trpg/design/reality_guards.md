# 现实世界护栏（Guards）清单｜避免跑团失控

> 用途：当玩家反复偏离或做高风险行为时，用“现实会如何反应”把局势拉回可控范围；每个护栏都应具备：触发条件 → 可见信号 → 合理后果 → 回到主线的替代路径。

## 1) 执法与秩序（警察/城管/站方安保）
- 触发条件：纠缠他人、尾随、阻碍通行、聚集围观、进入非公众区域、扰乱秩序、可疑拍摄。
- 可见信号：劝离、口头警告、要求出示证件、要求删除/停止拍摄、带到值班室登记。
- 合理后果：行政处置/口头训诫/登记留档；短期禁入点位；与单位/家属联系核验。
- 回主线替代：改做公开信息核对（公告/施工牌）、改为离场后远程反查、把记录脱敏交给第三方（互助群/媒体不直接点名）。

## 2) 社会视线（路人/商户/旁听者的传播链）
- 触发条件：反复问同类问题、长时间蹲守、与多人攀谈、行为与“普通通勤”不匹配。
- 可见信号：被盯、被拍、被议论、被商户婉拒；微信群/小红书/本地群出现“有人在打听”的提醒。
- 合理后果：信息扩散到“维护秩序方”耳朵里，触发更快的盘问/限制；线索窗口加速封存。
- 回主线替代：降低可见度（一次只问一个点）、改变时空位置（换站口/换天）、用“投诉/求助口径”而非追问真相。

## 3) 经济与资源（现金流/购买力/交通成本）
- 触发条件：长期蹲守、反复往返、临时住宿/交通支出、账号受限导致必须走人工渠道。
- 可见信号：余额不足、支付失败、需要补票/改现金；时间成本上升影响工作/生活。
- 合理后果：行动半径被迫缩小；只能选低成本路径；更依赖互助与公开信息。
- 回主线替代：把调查拆成“最小闭环任务”（只验证一个时间窗/一个公告抬头），把多点核对交给核对组分工。

## 4) 系统风控（账号验证/限流/实名核验）
- 触发条件：高频定位在敏感点位、与敏感话题关联的搜索/发帖、被举报、站内行为被注意。
- 可见信号：乘车码需验证、账号弹窗核验、发帖被限流、私信被提醒“涉嫌违规”。
- 合理后果：通行与传播能力下降；被迫走线下窗口；信息焦点被打断。
- 回主线替代：改用线下观察与公开牌照信息；把信息存本地、延迟发布；通过合法渠道提交投诉/信息公开申请（若适用）。

## 5) 组织流程（单位/学校/物业/社区）
- 触发条件：无视离职流程、旷工、在单位附近徘徊、与同事反复联系。
- 可见信号：HR/主管催办、发律师函式短信、门禁停用、保安问询。
- 合理后果：补偿受阻、背调风险、生活稳定性下降（迫使玩家权衡）。
- 回主线替代：把“保全生活”设为阶段目标（完成一项手续换取自由度），或选择“撤退保存线索”。

## 6) 医疗与身心（疲劳/失眠/情绪失控）
- 触发条件：熬夜蹲守、饮食紊乱、冲动对抗、连续受挫。
- 可见信号：注意力下降、判断偏执、与人冲突概率上升。
- 合理后果：行动质量下滑；更容易做出触发执法/风控的异常行为。
- 回主线替代：强制休整窗口（吃饭/睡觉/离场），用“脱敏整理记录”作为继续推进的低风险行动。

## 7) 物理空间（门禁/围挡/施工/监控盲区的反制）
- 触发条件：试图靠近箱体/机房门/非公众区域；绕开围挡。
- 可见信号：围挡增设、临时封闭、保安站位变化、人流被引导。
- 合理后果：你“看不到/到不了”关键处；只能从公开材料与间接证词推进。
- 回主线替代：转为收集公开施工牌、责任单位、维护窗口日期（推进里程碑 M3）。

## 8) 信息环境（谣言治理/通稿/反向误导）
- 触发条件：公开传播、点名单位/个人、使用敏感关键词、传播未经核实内容。
- 可见信号：通稿统一口径、辟谣号集体下场、群被转移、当事人沉默。
- 合理后果：信息焦点降档（清晰→断续）；玩家被动走向“情绪对抗”。
- 回主线替代：坚持“双源验证”门槛；只发布“最小可公开版本”；把可核对的锚点交给更多人去核对。

## 9) 对手的软反制（非超自然：承包商/公关/法务）
- 触发条件：你接近责任链条（工单编号、维护队伍、落款单位）。
- 可见信号：警告短信、商户被“提醒别乱传”、你被温和驱离。
- 合理后果：风险复利加速；你被逼离线下点位；对方以流程优势压制。
- 回主线替代：用公开信息反查链条、分散核对、保存证据链而非“当场对抗”。
